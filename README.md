# CredResolve AI Agent

This project is an AI-powered assistant that helps users discover relevant government schemes
based on their query and age.

## Tech Stack
- Python
- Sentence Transformers
- FAISS
- Gradio

## How it works
1. User enters a query and age
2. Semantic search retrieves relevant schemes
3. Eligibility logic is applied
4. Result is shown via Gradio UI
